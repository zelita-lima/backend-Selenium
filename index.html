Documentação da API

Bem-vindo à documentação da API.

Endpoints

  • GET /servidores/listar

    Obter uma lista de todos os servidores.

    Método: GET

    Exemplo:

    /servidores/listar

  • GET /servidores/listarServidor/:id

    Obter servidor pelo ID.

    Método: GET

    Exemplo:

    /servidores/listarServidor/1

  • POST /servidores/inserir

    Criar um novo servidor.

    Método: POST

    Exemplo:

    /servidores/inserir

    Corpo da Requisição:

    {
  "nome": "João da Silva",
  "cargo": "Engenheiro",
  "departamento": "Engenharia"
}

  • PUT /servidores/atualizar/:id

    Atualizar servidor existente.

    Método: PUT

    Exemplo:

    /servidores/atualizar/1

    Corpo da Requisição:

    {
  "nome": "João da Silva",
  "cargo": "Engenheiro",
  "departamento": "Engenharia"
}

  • DELETE /servidores/deletar/:id

    Excluir servidor pelo ID.

    Método: DELETE

    Exemplo:

    /servidores/deletar/1
